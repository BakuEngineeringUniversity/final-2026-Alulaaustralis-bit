# 🍽️ Recipe App

## Layihənin Strukturu
- `Recipe.kt` — Abstract sinif
- `BakingRecipe.kt` — Recipe + CookingMethod
- `VegetarianRecipe.kt` — Vegetarian resept
- `NonVegetarianRecipe.kt` — Ət resepti  
- `CookingMethod.kt` — Interface
- `RecipeSearch.kt` — Axtarış sinifi
- `Main.kt` — Əsas proqram + interaktiv menyu

## Branch-lar
- `first_branch` — Əsas layihə kodu
- `feature/menu` — İnteraktiv CLI menyusu əlavə edildi
- `main` — Əsas branch (merge edilmiş)

## Necə işlədilir
IntelliJ IDEA-da `Main.kt` faylını işlət.
